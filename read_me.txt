║  SECURITY BEST PRACTICES - TAILS EDITION                                              ║
╠═══════════════════════════════════════════════════════════════════════════════════════╣
| Tails OS is Amnesic                                                                   |
| The primary feature of Tails OS is that it is amnesic, meaning it leaves no trace on  |
| the host system once you shut down. This is a key element in maintaining privacy, as  |
| it ensures no personal data or activity is retained between sessions. While Tails does|
| offer the option to enable persistent storage on the USB, doing so can introduce      |
| risks, as it stores data on the USB drive that could potentially be recovered. Only   |
| enable persistent storage if absolutely necessary, and only if you fully understand   |
| the risks involved. If you're new to Tails, it is recommended to avoid persistent     |
| storage.                                                                              |
|                                                                                       |
| Minimize Your Digital Footprint                                                       |
| Tor is one of the Tails OS core features for ensuring anonymity. Every time you       |
| connect to the internet, Tor routes your traffic through multiple layers of encryption|
| across a decentralized network. To maintain privacy, make sure you do not disable Tor,|
| and avoid using non-Tor browsers or communications tools while in a Tails session.    |
| Refrain from accessing personal accounts unless using an anonymous identity to        |
| prevent linking your activities back to you.                                          |
|                                                                                       |
| Use Secure Communication Tools                                                        |
| When communicating online using Tails OS, always use encrypted, anonymous methods.    |
| Tails comes with several pre-configured, secure tools, such as Thunderbird with       |
| Enigmail for email encryption, and Pidgin with OTR (Off The Record) for instant       |
| messaging. Ensure that you do not use untrusted or unencrypted channels that could    |
| expose your identity.                                                                 |
|                                                                                       |
| Be Careful with Identifiers                                                           |
| Ensuring your online activities remain untraceable is key to maintaining privacy. To  |
| avoid linking your actions back to you, refrain from using the same username, email   |
| address, or login credentials across different platforms. Always use unique           |
| identifiers for each activity to prevent "cross-linking" your online presence. This is|
| especially important when engaging with forums, social media, or any other web service|
| while using Tails.                                                                    |
|                                                                                       |
| Tails is NOT a daily driver OS                                                        |
| While Tails OS is a powerful tool for privacy and anonymity, it is not designed to be |
| a daily-use operating system. Tails is intended primarily for situational use where   |
| privacy and security are of the utmost importance.                                    |
